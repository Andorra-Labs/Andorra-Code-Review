OpenCodeReview Setup

Define LLM providers and assign models to scanners + an arbiter for multi-LLM review. Generates a config.json the ocr CLI can read.

Import existing config (optional)

Load an existing config.json to edit and re-export.

Providers

Each provider is an endpoint (URL + protocol + key) and a list of model ids hosted there.

API keys are saved in plaintext in the downloaded file. For CI use, enter an env placeholder such as ${env:OCR_ANTHROPIC_API_KEY}; blank keys work only when the chosen provider preset supplies its own environment variable fallback. For local use, chmod 600 ~/.opencodereview/config.json after saving.

Workers

Pick which (provider, model) handles each role.

Scanners

Each scanner independently reviews the diff. Add multiple to compare or cross-check.

All scanners run at once. Faster, but every scanner draws GPU/API quota concurrently.

Arbiter

Cross-checks scanner findings and produces the final review.
