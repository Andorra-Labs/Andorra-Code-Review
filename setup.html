OpenCodeReview Setup

Define LLM providers and pick an active provider/model. Generates a config.json the ocr CLI can read.

Import existing config (optional)

Load an existing config.json to edit and re-export.

Providers

Each provider is an endpoint (URL + protocol + key) and a list of model ids hosted there.

API keys are saved in plaintext in the downloaded file. For preset providers, leave the key blank and set the provider's documented environment variable. Custom providers require a real key here. For local use, chmod 600 ~/.opencodereview/config.json after saving.
